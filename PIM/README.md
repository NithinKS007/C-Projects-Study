# Project: Personal Information Manager (PIM)

This is a small C study project built to practice the core parts of C programming in a simple, hands-on way.

## What it covers

- User account registration and login
- Profile lookup
- File-based text storage

## Build

Compile both source files together:

```bash
gcc main.c user_system.c contact_system.c -o main
```

## Notes

- `users.txt` is used as the backing store for user data.
